# **App Name**: StyleSphere

## Core Features:

- User Authentication: Secure user registration and login using JWT and bcrypt.
- Product Browsing & Filtering: Browse products with filtering options for price, size, color, and category.
- Product Details Page: Detailed view of products with image gallery and key information.
- Shopping Cart & Checkout: Add products to cart and complete the purchase via Stripe integration.
- Wishlist: Save favorite products to a personal wishlist.
- Admin Dashboard: Admin interface to manage products, users, and orders. Secured by role-based authorization.
- Intelligent Product Descriptions: Generate descriptions using the Bard API based on keywords provided when the product is created, which functions as a tool during product creation in the admin panel.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke a sense of sophistication and trust.
- Background color: Light Lavender (#E6E6FA) to provide a soft, clean backdrop.
- Accent color: Royal Blue (#4169E1) for interactive elements and highlights.
- Headline font: 'Playfair', serif, for an elegant, high-end feel. Body Font: 'PT Sans', sans-serif.
- Use a consistent set of minimalist icons to represent different product categories and UI actions.
- Employ a grid-based layout for product listings and a clean, structured design for all pages.
- Subtle transitions and animations for a smooth and engaging user experience during navigation and interactions.
